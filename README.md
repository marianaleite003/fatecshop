  <h1>🛍️ FatecShop</h1>
  <p><strong>FatecShop</strong> é um projeto de e-commerce desenvolvido para fins acadêmicos, com o objetivo de praticar habilidades em HTML e CSS. O site simula uma loja virtual com páginas de produtos, carrinho de compras, informações institucionais e suporte ao cliente.</p>

  <h2>📄 Visão Geral</h2>
  <p>O projeto contém páginas estáticas que representam as seções principais de uma loja online:</p>
  <ul>
    <li><strong>index.html</strong>: Página inicial com destaques e promoções.</li>
    <li><strong>produtos.html</strong>: Lista de produtos disponíveis.</li>
    <li><strong>carrinho.html</strong>: Itens adicionados ao carrinho.</li>
    <li><strong>ajuda.html</strong>: Perguntas frequentes e suporte.</li>
    <li><strong>sobre.html</strong>: Informações institucionais.</li>
  </ul>

  <p>Arquivos CSS correspondentes a cada página incluem:</p>
  <ul>
    <li>index.css</li>
    <li>produtos.css</li>
    <li>carrinho.css</li>
    <li>ajuda.css</li>
    <li>sobre.css</li>
  </ul>

  <h2>🧰 Tecnologias Utilizadas</h2>
  <ul>
    <li>HTML5</li>
    <li>CSS3</li>
  </ul>

  <h2>🚀 Como Visualizar o Projeto</h2>
  <p>Siga os passos abaixo para visualizar o site localmente:</p>
  <pre><code>git clone https://github.com/marianaleite003/fatecshop.git
cd fatecshop</code></pre>
  <p>Abra o arquivo <code>index.html</code> no seu navegador preferido.</p>

  <h2>📌 Observações</h2>
  <ul>
    <li>Este é um projeto estático, sem funcionalidades dinâmicas ou backend.</li>
    <li>É destinado apenas à demonstração visual de uma loja virtual.</li>
  </ul>
